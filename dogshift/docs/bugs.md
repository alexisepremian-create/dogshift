# Bugs known + fix history

This file used to hold every bug fiche as one big document. The bug
history has been split into one file per bug under [`docs/bugs/`](./bugs/)
so each fiche can be opened, linked, and read in isolation.

→ **Start here:** [`docs/bugs/README.md`](./bugs/README.md)

Convention:
- Each bug fiche is named with a kebab-case slug describing the symptom.
- Each fiche records: symptom, root cause, fix, related PRs, and explicit
  "do NOT do this again" warnings learned from prior attempts.
- Read the relevant fiche BEFORE attempting a fix in the same area — most
  of these have recurred at least once.
